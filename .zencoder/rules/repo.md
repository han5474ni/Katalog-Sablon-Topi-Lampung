# Repository Quick Reference

- **Framework**: Laravel 12.x (PHP 8.2+)
- **Frontend Stack**: TailwindCSS 3.x, Alpine.js/Livewire, Vite 6.x, Bootstrap 5.3+
- **Key Paths**:
  - `resources/views/admin/` — Admin Blade templates (product management forms live here)
  - `resources/css/admin/` — Custom CSS for admin area (e.g., modern-add-product.css)
  - `resources/js/admin/` — JavaScript logic for admin UI (product-management.js)
- **Build Tools**:
  - Tailwind configured via `tailwind.config.js`
  - Assets compiled with Vite (`npm run dev` / `npm run build`)
- **Testing / Dev Commands**:
  - `composer test` or `phpunit` (PHP tests)
  - `npm run lint` and `npm run build`
- **Environment Notes**:
  - `.env` configures DB (MySQL default) and Google OAuth keys
  - Storage link required: `php artisan storage:link`
- **Product Management UI**:
  - Blade view: `resources/views/admin/management-product.blade.php`
  - Styling partial: `resources/css/admin/modern-add-product.css`
  - Interactions: `resources/js/admin/product-management.js`
- **Known UI Goal**: Ensure add-product form matches modern Tailwind-inspired design.
