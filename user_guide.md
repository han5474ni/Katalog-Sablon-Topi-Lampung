# User Guide: Katalog Sablon Topi Lampung

Panduan ini ditujukan untuk admin/customer admin dalam mengelola sistem e-commerce sablon topi.

## 1. Login Admin
- Akses halaman login admin: `/admin/login`
- Masukkan email & password admin.
- Jika lupa password, gunakan fitur reset password (jika tersedia).

## 2. Dashboard & Navigasi
- Setelah login, Anda akan diarahkan ke dashboard admin.
- Navigasi utama: Pesanan, Produk, Custom Design, User Management, Analytics, Pengaturan.

## 3. Manajemen Pesanan
- Lihat daftar pesanan masuk di menu "Pesanan".
- Klik detail pesanan untuk melihat info lengkap, status, dan file desain.
- Ubah status pesanan (approve, reject, complete) sesuai proses produksi.
- Gunakan fitur upload bukti pembayaran jika diperlukan.

## 4. Custom Design
- Menu "Custom Design" untuk mengelola pesanan custom.
- Approve/reject desain custom yang diupload customer.
- Atur harga custom design di menu "Custom Design Prices".

## 5. Manajemen Produk
- Tambah/edit/hapus produk di menu "Produk".
- Atur stok, harga, dan gambar produk.

## 6. User Management
- Kelola data user/customer di menu "User Management".
- Reset password user jika diperlukan.

## 7. Analytics & Laporan
- Lihat laporan penjualan, status pesanan, dan performa di menu "Analytics".
- Filter data berdasarkan tanggal.

## 8. Pengaturan Sistem
- Atur konfigurasi chatbot, notifikasi, dan integrasi lain di menu "Pengaturan".

## 9. Notifikasi
- Sistem mengirim notifikasi otomatis ke customer via email/WhatsApp sesuai status pesanan.

## 10. Troubleshooting
- Jika ada error, cek menu "Logs" atau file `storage/logs/laravel.log`.
- Pastikan konfigurasi email & storage sudah benar.
